# Support Ticket Analysis

## Project Overview
This project analyzes customer support ticket data to identify trends in **resolution times**, **customer satisfaction** and **survey results**. Key questions include:
1. Average resolution time by priority.
2. Top agents by customer satisfaction.
3. Relationship between agent interactions and survey results

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Visual Studio Code

## Project Structure

data/: Contains datasets.
scripts/: Python scripts for analysis.
images/: Visualizations.
results/: Summary outputs.
